UI日历组件

组件引用地址

http://s1.56img.com/script/ui/calendar/v1/calendar_v1.min.js

组件参数

setDate
{Date} - 可选, 当前选中的日期（月份是从0开始），如果设置了，日历会弹出选中日期这个月份，没有的话，就现在当前月份
minDate
{Date} - 可选, 日历的最小日期, 默认不开启
maxDate
{Date} - 可选, 日历的最大日期, 默认不开启
popup
{Boolean} - 可选, 日历是否为弹出,默认为false, 不开启
target
[string,dom] - 可选, 插入或者触发弹出日历的标签id
onSelect
{function} - 可选, 选中一个日期执行的回调函数
dateFormat
{string}[default,fullDate] - 可选, 输出日期的格式

组件可调用方法

hide()
如果日历是弹出形式, 默认是隐藏日历
show()
显示日历
selectDate({Date})
选中一个新的日期
getDate([fullDate])
获取日期，日期格式2000/1/1, fullDate参数可选，输出日期格式2000年1月1日

场景一：在指定id的标签插入日历

需要设置target参数，如果没有直接append body里

例子代码：

var cale = new Calendar({
	target: "demo"
});

场景二：有日期范围的日历

例子代码：

var cale = new Calendar({
	minDate :   new Date(2009,9,2),
	maxDate :   new Date(2011,11,16),
	target: "demo"
});

场景三：指定日历的日期

例子代码：

var cale = new Calendar({
	setDate : new Date(2012,11,5),
	target: "demo"
});

场景四：灵活运用日历插件公有方法

例子代码：

var cale = new Calendar({
	setDate : new Date(2012,11,5),
	target: "demo"
});
cale.show();
cale.hide();
cale.selectDate(new Date(2012,11,30));
cale.getDate();

场景五：点击文本框里弹出日历

例子代码：

var cale = new Calendar({
	target: "demo",
	popup:true
});

场景六：点击标签弹出日历，选择日期页面跳转

例子代码：

var cale = new Calendar({
	target: "demo",
          popup:true,
          onSelect:function(){
              alert(this.getDate());
          }
});
